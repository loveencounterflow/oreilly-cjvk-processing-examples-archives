#	General notes:

#       to Unicode.  The mapping data comes from The Unicode Standard,
#	Volume 1, with character names from Unicode 1.1.  Thus this
#	data also applies to ISO/IEC 10646-1:1993.

#	The mappings for Adobe Expert "Character Set" have not been
#	added to this table yet.
#
#	Format:  Three tab-separated columns
#		 Column #1 is the Adobe "Character" Name
#		 Column #2 is the Unicode (in hex as 0xXXXX)
#		 Column #3 the Unicode name (follows a comment sign, '#')
#
#	The entries are in (ASCII) alphabetical order
#
#	Any comments or problems, contact <glenn@metis.com>
